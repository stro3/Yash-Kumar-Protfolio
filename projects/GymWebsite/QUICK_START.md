# Quick Start Guide

Hey! This is super easy to use.

## Start Your Website

Just double-click: **start.bat**

That's it! Wait a few seconds and then open your browser to:
**http://localhost:5000**

## What You'll See

- Your gym website homepage
- Login and signup pages  
- Membership plans
- Classes and trainers
- Contact page

## Need Help?

Everything runs on one server now. No need to start frontend and backend separately.

If the port 5000 is already in use, the server won't start. Just close whatever is using port 5000 and try again.

Enjoy! 💪
